Call to Action Button

Call to Action Button

Select "Yes" and this will add a Call to Action button on your Slide.

Button Color

Select a color for the button. If you leave this blank it will default to the Theme's highlight color.

Button Text

Enter some text to display on your button, such as "Read More" or "Learn More".

Button Text Color

Select your button text color.

Button Link

Enter the URL for your button, such as the permalink to your blog page. You must enter a URL for the Call to Action button to work properly.
