Section Layout

There are four different layouts to choose from for your Section: Right, Left, Center, and Custom.

Left and Right

If your Section has both an image and content then selecting Right aligns the content to the right, Left aligns the content left. If there is only content and no featured image or vice versa, it automatically places it in the center of the Section.

Center

Choosing Center places both the content and image in the center of the Section.

Custom

A Custom layout removes the Title and Call to Action button from the Section, leaving only the content or images to display. If you want a Section that displays only three columns of content, for example, then you can select the Custom layout and use the Column Shortcodes in the WYSIWYG editor.
